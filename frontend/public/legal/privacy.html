Политика конфиденциальности

Последнее обновление: 13 февраля 2026

ParkingSmart (далее — «Сервис») разработан с уважением к вашему праву на приватность. Мы собираем только те данные, которые необходимы для работы сервиса, и не передаём их третьим лицам.

Кратко: Мы не продаём ваши данные. Мы не показываем рекламу. Ваш номер телефона надёжно зашифрован и виден только тем, кто физически находится рядом с вашим автомобилем и ищет блокирующий выезд.

1. Какие данные мы собираем

1.1. Номер телефона
Единственный обязательный идентификатор. Используется для создания уникального аккаунта и для связи между водителями. Номер хранится в зашифрованном виде (AES-256). Для поиска используется необратимое хэширование (SHA-256).

1.2. Геолокация
Координаты места парковки, которые вы указали сами. Мы не отслеживаем ваше местоположение в фоновом режиме. Только та точка, куда вы поставили маркер.

1.3. Статус блокировки
Отметка «Блокирую» или «Не блокирую», которую вы делаете сознательно при парковке.

1.4. История звонков
Фиксация факта звонка (кто звонил, кому, когда) для отображения статуса «Уже звонили». Номера вызываемого абонента не хранятся в открытом виде.

2. Цели обработки данных

  • Идентификация пользователя (номер телефона → хэш)
  • Поиск блокирующих автомобилей в радиусе 4 метров
  • Отображение зашифрованного номера для звонка
  • Предотвращение повторных звонков тому же водителю
  • Автоматическое удаление устаревших данных

3. Правовые основания

Обработка персональных данных осуществляется на основании:

  • ст. 6 Федерального закона «О персональных данных» № 152-ФЗ
  • Вашего согласия, выраженного при регистрации (чекбокс «Даю согласие»)

4. Сроки хранения

  • Парковки: 48 часов — автоматическое удаление по истечении срока
  • Звонки: 7 дней — для отображения истории
  • Аккаунт пользователя: бессрочно, пока вы не удалите его самостоятельно (напишите в поддержку)

5. Шифрование и защита

  • Номер телефона шифруется алгоритмом AES-256-CBC с уникальным вектором инициализации для каждого пользователя
  • Поиск осуществляется по необратимому хэшу (SHA-256)
  • В интерфейсе номер отображается только в формате +7 *** *** XX XX
  • Все соединения — HTTPS
  • База данных изолирована в приватной сети Docker

6. Передача данных третьим лицам

Мы не передаём персональные данные третьим лицам. Исключение — запросы уполномоченных государственных органов в соответствии с законодательством РФ.

Геоданные обрабатываются с использованием API Яндекс.Карт. Использование регулируется Пользовательским соглашением Яндекс.Карт.

7. Трансграничная передача

Все серверы ParkingSmart физически расположены на территории Российской Федерации. Трансграничная передача данных не осуществляется.

8. Ваши права

  • Уточнять, какие данные о вас хранятся
  • Требовать удаления аккаунта и всех связанных данных
  • Отозвать согласие на обработку персональных данных

Для реализации прав напишите на privacy@parkingsmart.ru. Мы обязаны ответить в течение 30 дней.
